Grid

A simple and fun game where kids learn the concepts of coding at the same time. The goal is to code the duck to get from the starting point to the end lilypad while avoiding the cat tails.

Our goal at Coding Ducks is teach kids the fundamentals of coding through fun and interactive games instead of the traditional boring way.
